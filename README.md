# GL3-Asymptotic-Large-Sieve
Mathematica implementation for GL3 Asymptotic Large Sieve

This repository contains the Mathematica notebooks used for the computations involving Hecke algebras in the paper.

## Notebooks

### GL3Ratio.nb

This carries out the "ratios" calculations in Section 13 of the paper.

### UrInf.nb

This computes and bounds the infinite Euler product of $\mathfrak{A}_{p}$'s in Section 12.5 of the paper.

### UrFinExact.nb

This is used to compute the finite Euler product $\mathcal{E}$ over $\mathfrak{k}_{i}'$ in the sum $\mathcal{U}^{(r)}(h,k)$; see Section 12.4 of the paper.
